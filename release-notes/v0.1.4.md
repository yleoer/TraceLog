# TraceLog v0.1.4

优化高数据量性能、更新检查频率、Markdown 渲染和时间选择体验。

## 新增

- Issue 和临时需求列表分页。
- 时间选择器快捷时间：上班 08:00、上午 10:00、中午 12:00、下班 17:00。
- 应用更新检查结果缓存和上次检查时间展示。

## 优化

- 数据库新增标签索引表、搜索索引引用表和多组高频查询索引。
- 列表搜索改用 FTS 与索引表，提升高数据量下的查询表现。
- 启动时自动补齐性能相关 schema、标签索引和搜索索引引用。
- dev 版本跳过最新版本检查，避免开发环境频繁请求 GitHub。
- 快捷时间两行展示，选择日期后只调整时间，不再把日期重置为当天。

## 修复

- 修复 Markdown 列表、标题、引用、分割线和表格等内容显示样式。
